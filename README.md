*   **“一级助推器”信号 (时序逻辑)**:
    1.  **动量反转**: MACD指标在近期发生过**“水下金叉”**。
    2.  **位置确认**: 股价在近期经历过对中期均线（如60日线）的**回踩和企稳**。
    3.  **强者基因**: 个股自身的中期趋势强度（RPS120）依然保持在强势区域（如 > 75）。
